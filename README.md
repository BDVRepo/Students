# Проект Students - Лабораторная работа по Windows Forms

## Описание

Проект представляет собой Windows Forms приложение для работы с базой данных студентов. Реализованы формы для просмотра и редактирования данных о студентах с использованием масок ввода, выпадающих списков и привязки данных.

## Требования

- **Visual Studio** 2010 или выше (рекомендуется 2019)
  - Скачать: https://visualstudio.microsoft.com/ru/free-developer-offers/
  - Или старую версию (2010): http://ui.chuvsu.ru/soft/Microsoft/MSDN/VS2010ExpressRUS.iso

- **MS SQL Server** 2008 R2 Express Edition или выше (рекомендуется не выше 2014)
  - Скачать: http://ui.chuvsu.ru/soft/Microsoft/MSDN/SQLEXPRWT_x86_RUS.exe
  - Или с сайта Microsoft: https://www.microsoft.com/ru-ru/sql-server/sql-server-downloads

- **SQL Server Management Studio** (SSMS)
  - Входит в состав SQL Server или скачивается отдельно

- **.NET Framework 4.7.2** или выше

## Установка и настройка

### 1. Установка SQL Server

1. Установите MS SQL Server с настройками по умолчанию
2. При установке выберите режим аутентификации "Смешанный режим"
3. Установите пароль для пользователя `sa`: `MSSQL`
4. Запомните имя сервера (обычно `localhost` или `localhost\SQLEXPRESS`)

### 2. Создание базы данных

1. Запустите **SQL Server Management Studio**
2. Подключитесь к серверу:
   - **Имя сервера:** `localhost` (или имя вашего сервера)
   - **Проверка подлинности:** SQL Server Authentication
   - **Имя пользователя:** `sa`
   - **Пароль:** `MSSQL`

3. Создайте базу данных:
   - Правой кнопкой мыши на "Базы данных" → "Создать базу данных..."
   - Имя: `students`
   - Нажмите "ОК"

4. Выполните SQL скрипт:
   - Правой кнопкой мыши на базе данных `students` → "Создать запрос"
   - Откройте файл `Database/create_database.sql`
   - Скопируйте содержимое в окно запроса
   - Нажмите "Выполнить" (F5)
   - Должно появиться сообщение "Выполнение команд успешно завершено"

5. Проверьте создание таблиц:
   - В обозревателе объектов разверните `students` → "Таблицы"
   - Должны быть созданы таблицы: Предметы, Специальности, Студенты, Оценки

### 3. Настройка проекта

1. Откройте проект в Visual Studio:
   - Запустите Visual Studio
   - Файл → Открыть → Проект/Решение
   - Выберите файл `Students.sln`

2. Настройте строку подключения (если необходимо):
   - Откройте файл `Students/App.config`
   - Проверьте строку подключения:
     ```xml
     <connectionString>Data Source=localhost;Initial Catalog=students;User ID=sa;Password=MSSQL</connectionString>
     ```
   - Если имя сервера отличается, измените `localhost` на имя вашего сервера

3. Восстановите зависимости (если требуется):
   - Правой кнопкой мыши на решении → "Восстановить пакеты NuGet"
   - Или: Сборка → Перестроить решение

## Запуск проекта

1. В Visual Studio нажмите F5 или кнопку "Пуск"
2. Откроется главная форма приложения
3. Нажмите кнопку "Таблица 1" для открытия формы студентов
4. Используйте панель навигации или кнопки справа для работы с записями

## Функциональность

### Главная форма (Form1)
- Отображение названия базы данных и фамилии
- Кнопки для открытия различных таблиц

### Форма студентов (Form2)
- **Панель навигации:** стандартная панель для перемещения по записям
- **Поля данных:**
  - Текстовые поля (ФИО, Адрес, Группа)
  - Маски ввода (Телефон, Паспорт, Номер зачетки)
  - Выпадающие списки (Пол, Родители, Специальность)
  - Числовой счетчик (Курс)
  - Выбор даты (Дата рождения, Дата поступления)
  - Чекбокс (Очная форма обучения)
- **Кнопки навигации:** программное управление записями
- **Операции:** добавление, редактирование, удаление, сохранение

## Возможные проблемы и решения

### Проблема: Не удается подключиться к базе данных

**Решение:**
1. Проверьте, запущен ли SQL Server:
   - Панель управления → Администрирование → Службы
   - Найдите "SQL Server (MSSQLSERVER)" или "SQL Server (SQLEXPRESS)"
   - Убедитесь, что служба запущена

2. Проверьте строку подключения в `App.config`

3. Проверьте имя сервера:
   - В SQL Server Management Studio посмотрите имя сервера в обозревателе объектов
   - Используйте это имя в строке подключения

### Проблема: Ошибка при выполнении SQL скрипта

**Решение:**
1. Убедитесь, что база данных `students` создана
2. Выполняйте скрипт по частям (создание таблиц, затем INSERT)
3. Проверьте, что используется правильная база данных: `USE students;`

### Проблема: Ошибки компиляции в Visual Studio

**Решение:**
1. Восстановите зависимости: Сборка → Перестроить решение
2. Проверьте версию .NET Framework (должна быть 4.7.2 или выше)
3. Убедитесь, что все файлы проекта включены в решение
